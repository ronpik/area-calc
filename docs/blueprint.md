# **App Name**: AreaCalc

## Core Features:

- Location Display: Display current location on a map using device location services.
- Location Recording: Record the current location upon button press, and add it to the polygon.
- Polygon Display: Display the created polygon on the map.
- Area Calculation: Calculate and display the area of the polygon in square meters.
- Accuracy Indicator: Indicate the accuracy of location estimation using a color-coded button (Red, Orange, Green).
- Threshold Configuration: Configure the accuracy level boundaries using thresholds (e.g., in meters).

## Style Guidelines:

- Primary color: A calm, nature-inspired blue (#6699CC) evoking trust and precision.
- Background color: Light grayish-blue (#F0F4F7), to provide a clean, uncluttered backdrop that subtly complements the primary color.
- Accent color: A warm, contrasting orange (#E6734D) to highlight interactive elements, drawing user attention.
- Font: 'Inter', a sans-serif font suitable for both headlines and body text, chosen for its modern and neutral appearance to ensure legibility on mobile screens.
- Use clear, geometric icons for location and area-related functions.
- Prioritize a clean, map-centric interface to focus on location data.
- Implement subtle animations for button feedback and map updates.
